The Maze Project

Experience the thrill of navigating through a 3D maze game!

Key Features

3D Raycasting Engine

Our game features a powerful 3D raycasting engine that renders the maze environment in real-time, providing an immersive experience.

Dynamic Camera Movement

Navigate the maze with smooth and responsive camera controls, enhancing your gameplay experience.

About the Project

This project was inspired by classic 3D games and a passion for game development. The idea was to create an engaging and visually appealing game that combines the challenge of a maze with the excitement of real-time rendering.

The project is part of my portfolio for Holberton School, showcasing my skills in software development and game design.
