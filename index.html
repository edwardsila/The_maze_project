About the Project

This project involves creating a 3D maze game using raycasting and SDL2. The game includes features such as collision detection, texture mapping, and user interaction for movement and rotation.

Features

  • 3D Raycasting Engine
  • Dynamic Camera Movement
  • Collision Detection
  • Texture Mapping
  • Interactive User Controls
